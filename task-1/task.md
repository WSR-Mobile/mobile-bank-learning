# Задание 1. Главный экран мобильного банка

## Цель

Сверстать главный экран учебного мобильного приложения «Мой Банк» по макету в Figma.

## Макет

Ссылка на Figma: https://www.figma.com/design/fExldtRJ43YXRPPid3rvIs

Макет должен быть опубликован с доступом «Anyone with the link can view», чтобы его мог открыть любой студент без входа в Figma.

Экран необходимо повторить максимально близко к макету: отступы, размеры, цвета, шрифты, скругления и расположение элементов.

## Что должно быть на экране

- приветствие пользователя;
- иконка уведомлений;
- блок с общим балансом;
- кнопка с иконкой глаза для показа и скрытия баланса;
- карточка банковской карты;
- блок быстрых действий: «Перевести», «Оплатить», «Пополнить»;
- раздел «Последние операции» с несколькими статичными операциями;
- нижняя навигация.

## Логика

Реализуйте только одно действие: при нажатии на иконку глаза сумма баланса должна скрываться и отображаться снова при повторном нажатии.

Все данные на экране должны быть захардкожены: не используйте API, базу данных и авторизацию.

## Требования

- экран должен корректно выглядеть на устройстве или эмуляторе;
- интерфейс должен быть адаптирован под разные размеры экранов, насколько это возможно;
- повторяющиеся элементы оформите в виде переиспользуемых компонентов;
- используйте понятные названия файлов, переменных и компонентов.

## Результат

Исходный код главного экрана и короткое видео или скриншоты работающего приложения, включая состояние со скрытым балансом.
