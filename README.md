# Мой Банк — учебные задания

Репозиторий содержит последовательность заданий для обучения мобильной разработке на примере учебного банковского приложения «Мой Банк».

Задания выполняются по порядку. На первых этапах не используются API, база данных и авторизация: данные задаются локально в коде.
